# Deep Learning Models from Scratch (Work in Progress)

This repository contains my implementations of different neural network architectures built from scratch as part of my deep learning learning journey.

I am currently developing this repository to gain strong conceptual understanding of Deep Learning by implementing models manually using PyTorch and TensorFlow instead of only using high-level APIs.

## Current Focus

- Artificial Neural Networks (ANN)
- Convolutional Neural Networks (CNN)
- Recurrent Neural Networks (RNN)
- Long Short-Term Memory (LSTM)
- Transformers (learning phase)
- Generative AI (learning phase)

## Frameworks Used

- PyTorch (main framework)
- TensorFlow 2.0 / Keras
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## What this repository includes

- Building neural networks from scratch
- Training models on different datasets
- Code with comments for learning and revision
- Notes added inside code for better understanding
- Experiments with different architectures

## Status

This repository is currently under development and will be continuously updated as I learn and implement more deep learning models.

## Purpose

The goal of this repository is to strengthen my understanding of Machine Learning, Deep Learning, NLP, and Transformers by implementing models step-by-step and maintaining them for future reference.
